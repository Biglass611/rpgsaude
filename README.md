📱 RPGSaude — Aplicativo de Chat em Grupo com Gamificação de Atividades Físicas

O RPGSaude é um aplicativo mobile criado para incentivar atividades físicas de forma divertida e colaborativa. Ele funciona como um chat de grupos, onde os usuários podem interagir com amigos, criar avatares personalizados e competir entre si por meio de desafios e recompensas.

Este projeto demonstra boas práticas no desenvolvimento de aplicativos móveis utilizando React Native, integrado a um backend robusto desenvolvido em Java Spring Boot com autenticação via JWT Security Token.

✨ Descrição do Projeto

O RPGSaude combina elementos de RPG e gamificação para motivar hábitos saudáveis.
Os usuários podem criar grupos, conversar em tempo real, e registrar suas atividades físicas para subir de nível, desbloquear itens e competir.

O sistema possui:

Fluxo de autenticação completo usando JWT;

Gerenciamento de grupos e avatares;

Chat integrado;

Sistema de desafios, recompensas e progresso.

🧩 Funcionalidades

💬 Chat em Grupo:
Converse com amigos, compartilhe progresso e organize desafios.

🛡️ Criação de Grupos:
Crie grupos personalizados para competir entre si.

🧙‍♂️ Avatares Personalizáveis:
Cada usuário pode criar e evoluir seu personagem conforme pratica atividades físicas.

🏃 Registro de Atividades Físicas:
Cada tarefa concluída gera atributos e recompensas.

🏅 Sistema de Recompensas:
Competições saudáveis entre amigos baseadas em exercícios e conquistas.

🔐 Autenticação Segura:
Login e validação de sessão via JWT, garantindo acesso seguro às rotas protegidas.

🛠️ Tecnologias Utilizadas

Backend

Java Spring Boot

Maven

JWT Security Token
